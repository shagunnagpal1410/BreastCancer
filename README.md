# 🩺 Breast Cancer Detection using Machine Learning

A Machine Learning-powered web application that predicts whether breast cancer is detected based on various tumor characteristics. The model is trained on diagnostic features and deployed using Flask, allowing users to receive real-time predictions through a simple web interface.

## 🚀 Live Demo

🔗 **Application Link:** https://breastcancer-zj6t.onrender.com/predict

---

## 📌 Overview

Breast cancer is one of the most common cancers worldwide. Early detection can significantly improve treatment outcomes. This project leverages Machine Learning techniques to analyze tumor characteristics and predict whether breast cancer is present.

The application accepts medical measurements as input and returns one of the following predictions:

- ✅ Breast Cancer Detected
- ✅ Breast Cancer not Detected

## 🧠 Model Workflow

1. User enters tumor characteristics.
2. Flask receives input data.
3. Input is converted into a NumPy array.
4. Trained ML model generates prediction.
5. Result is displayed on the webpage.

According to your Flask application:

```python
prediction = model.predict(final_features)
output = 'Breast Cancer Detected' if prediction[0] == 1 else 'Breast Cancer not Detected'
```
